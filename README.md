> Mobile attendance app with AI Face Recognition & GPS Geofencing

- Built with **Flutter** (mobile) + **Laravel 11** (backend API) + **Python Flask** (face recognition engine)
- Face recognition using **FaceNet / DeepFace** with cosine similarity matching
- GPS Geofencing with **Haversine Formula** for location verification

- **Stack:** Flutter · Laravel 11 · Python Flask · MySQL · DeepFace
